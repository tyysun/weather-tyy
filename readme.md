# weather-tyy
一个查询天气的命令行工具,剔除了多余的信息,只保留实时温度,本日温度区间等重要信息,更简洁,更明了

## 安装本工具

- 在命令行中输入:
```
 npm install -g weather-tyy 
```

## 使用本工具

- 输入命令:
```
 weather                // 查询IP地址对应的所在地天气
```

```
 weather 西安           // 查询具体城市天气
```
